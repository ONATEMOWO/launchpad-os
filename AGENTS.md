# AGENTS.md

This repository is a senior seminar CS project called LaunchPad OS.

## Project Goal
Build a student application workspace for internships, scholarships, and research opportunities.

## Priorities
- Keep code simple, readable, and explainable
- Prefer straightforward Flask patterns over clever abstractions
- Do not add unnecessary dependencies
- Preserve Bootstrap-based UI consistency
- Make small, focused changes
- Keep outputs easy for a student to explain in class and reports

## Core Concepts
- Users authenticate and access only their own data
- Opportunities track deadlines, statuses, categories, notes, and priorities
- Materials Vault stores reusable application content
- Requirements track completion state for each opportunity
- Dashboard summarizes upcoming work and progress

## Working Rules
1. Respect existing project structure
2. Avoid overengineering
3. Prefer stability over novelty
4. Summarize files changed after each task
5. Run checks when possible
