# Challenges Log

## Stage 1 Setup Challenges

1. Cookiecutter generation initially failed because the app name used hyphens instead of underscores, which is invalid for a Python package name.

2. The starter app first loaded without styling because frontend assets had not been built yet, causing 404 errors for CSS, JavaScript, and favicon files.

3. Git was initially initialized in the wrong directory once, which had to be corrected so the actual project folder became the repository root.

4. The VS Code shell command (`code`) was not available in the terminal, so command-line launching of VS Code was not immediately possible.

5. After installing Node dependencies and running the build process, the frontend assets loaded successfully and the app rendered correctly.
